About Me

I’m a software developer from Germany.

My main areas of interest are frontend and backend web application development. As a frontend developer I focus on creating accessible interfaces using semantic HTML and CSS. I use TypeScript (or JavaScript) with Vue, React or Angular to create clean and maintainable web applications. For backend development, my focus is on Java (for example using Spring Boot) and Rust. I always enjoy learning new technologies, or sharing knowledge with my peers. I use the pronouns they/them.

Highlighted Projects

  • YuGiOh Deck Tool - A Vue.js web application to manage Yu-Gi-Oh! decks.
  • Planning Poker - A java application to play planning poker.
  • Musicbrainz Enricher - A Java application to enrich MusicBrainz data from linked sources.
  • webmention4j - A Webmention implementation in Java.
  • musql - A music tags to SQL conversion tool written in Rust.
