Status update example

2025/10/06 -- David Frank

It is also possible to enable a different post style with this front matters:

+++
[extra]
is_status_update = true
+++

The full post will be shown, a nice way to give quick status update.

Webmentions and Webring support

2025/10/05 -- David Frank

To support POSSE sydnications, Marisa comes with its own fediverse comment support.

We did this with 2 services: Bridgy and Webmention.io, by signing up both, you can backfeed interactions back to your site as webmentions.

Marisa currently only show interactions counts, future work will bring more social graphs into context.

You can sign up for a good-old webring to help your site’s discovery. See config.toml for more information.
