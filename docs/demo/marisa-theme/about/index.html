Marisa is a non-opinionated theme with convenient features for personal sites and small indie teamsm, built for Zola.

#Intro

Do you ever think about the late-90s to early-00s when everyone has their own MySpace or GeoCities?

Do you enjoy styling your homepage with custom CSS? Like CSS Zen Garden, where the same HTML structures can turns out very different?

Marisa is a homage to that time.

#Demo

A live demo is available at theme.marisa.club.

#Features

  • Non-opinionated layout, bring your own styles;
  • Easy i18n integration, even create custom strings;
  • 2-style of posts: full-length articles or status updates;
  • Standard shortcodes for <audio> and <video>;
  • Table of content support;
  • Fuzzy search integration;
  • Fediverse interactions;
  • Optional javascript;
  • Vanilla CSS, no sass needed;
  • NoAI used;
  • Support Indieweb.

#Installation

  1. zola init your websites;

  2. git submodule add https://codeberg.org/bitinn/marisa-theme.git themes/marisa-theme install marisa theme;

  3. copy its config.toml to your root folder, then start editing.

#Update

  1. git submodule update --remote themes/marisa-theme
