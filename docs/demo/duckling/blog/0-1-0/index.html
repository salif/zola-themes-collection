Duckling is here!

Almost everything that Duckquill offers is available in Duckling in a more minimal design.

Additionally, the following features were added:

  • Project documentation! Find a demo under Docs.
  • Subprojects, a way to showcase a project’s features.

Learn how to use Duckling for your own project website in the docs.
