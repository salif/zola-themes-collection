Duckling

Duckling is a minimal and feature-rich Zola theme that has the purpose of greatly simplifying the process of rolling up your project website. It aims to provide all the necessary options for comfortable writing, while keeping the balance of simplicity.

It is based on the Duckquill theme for blogs.
