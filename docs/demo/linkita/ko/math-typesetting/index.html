      Linkita 테마는 KaTeX를 사용하여 $\LaTeX$ 수학 공식을 지원합니다.

      • KaTeX를 전체적으로 활성화하려면, 프로젝트 설정 파일에서 extra.math 파라미터를 true로 설정하세요.
      • 페이지별로 KaTeX를 활성화하려면, 콘텐츠 파일의 프론트매터에 extra.math = true 파라미터를 포함하세요.

      참고: 지원되는 TeX 함수 온라인 레퍼런스를 사용하세요.
