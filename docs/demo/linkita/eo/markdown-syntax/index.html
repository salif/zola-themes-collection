    Citaĵoj

    La citaĵa elemento reprezentas enhavon cititan el alia fonto, laŭvole kun citaĵo, kiu devas esti ene de footer aŭ cite elemento, kaj laŭvole kun entekstaj ŝanĝoj kiel komentoj kaj mallongigoj.

    Citaĵo sen atribuo

    Tiam, ad mint andaepu dandae nostion secatur sequo quae. Rimarku, ke vi povas uzi Markdown-sintakson ene de citaĵo.

    Citaĵo kun atribuo

    Ne komuniku per kunhavigo de memoro, kunhavu memoron per komunikado.
    — Rob Pike1
