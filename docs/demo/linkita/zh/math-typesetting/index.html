    Linkita 主题支持使用 KaTeX 排版 $\LaTeX$ 数学公式。

    • 要全局启用 KaTeX，请在项目配置文件中将参数 extra.math 设置为 true。
    • 要针对单个页面启用 KaTeX，请在内容文件的前言（frontmatter）中包含参数 extra.math = true。

    注意： 请参考 支持的 TeX 函数 在线文档。
