This website

This website doesn't collect or share any data.

No personal data, tracking data or any other type of data is leaving your device at all and so it can't even reach the Internet.

This website is made with Zola and it is licensed under the Free and Libre Open Source AGPL-3.0-or-later license. It doesn't track you and has no third party libraries included.

This website is hosted on Hetzner in Nuremberg, Germany, running on Debian with a Caddy web-server.

This website does not set or read any cookies or other locally stored data. When a visitor accesses this website, the web-server only logs a time-stamp, the accessed file and the HTTP status code. No IP-address or any other information that might identify the visitor is logged.

This policy is effective as of 21. May 2025. Any changes to this privacy policy can be found on Codeberg.
