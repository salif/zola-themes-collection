# Zola Themes Collection

[A curated directory of themes for Zola](https://salif.github.io/zola-themes-collection/)

## Theme requirements

Have a `theme.toml` file. Have a `comfig.toml` file or `themes/theme-name/config.toml`. No commited `public` directory.
